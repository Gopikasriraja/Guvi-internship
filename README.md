# GUVI Internship - Signup/Login/Profile

Simple project prepared for GUVI internship submission.

Steps to use:
1. Copy .env.example to .env and fill in your database credentials.
2. Initialize MySQL by running sql/init.sql on your MySQL server.
3. Deploy to Render (or another host).
4. Open index.html to register, login, and manage profile.

If you need step-by-step help to upload to GitHub and deploy to Render, ask me and I will guide you.
